Code Breaker Army Edition - NL versie
Download en installatie instructies:

1. Download het bestand "Code Breaker Army Edition.exe"
2. Rechtsklik op het gedownloade bestand
3. Kies "Eigenschappen"
4. Onderaan zie je: " Blokking opheffen (en nog tekst)"
5. Vink het vakje "Deblokkeren" aan
6. Klik op "Toepassen" en daarna op "OK"
7. Dubbelklik op het exe-bestand om de game te starten

Waarom deze stappen?
Windows blokkeert automatisch alle programma's die van internet worden gedownload als beveiligingsmaatregel. Door het bestand te deblokkeren geef je aan dat je het vertrouwt.
Systeemvereisten:
Windows 8/10/11 elke versie van deze windows is oke
Het moet 64 bit aankunnen ( de os hier boven kunnen dat).
